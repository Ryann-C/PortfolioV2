Mes projets


Voici les projets que j'ai pu effectué lors de ma formation en BTS SIO, que ce soit au lycée en initial et durant mes stages pendant ces 2 années. J'ai pu participer à des projets personnels mais aussi à un projet en groupe :


  • Projet Shop-Shop (En cours de construction)


    Le Projet Shop-Shop, est un projet individuel réalisé en classe. Ce projet à pour but de proposer un site e-commerce avec le thème de son choix.
    Langage : HTML, CSS, PHP, Twig, MySql

  • Projet Carbone (Prototype)


    Le Projet Carbone, réalisé en collaboration avec l'ensemble de la classe, vise à évaluer l'empreinte carbone générée par notre lycée. Son objectif principal est de quantifier cette empreinte carbone dans l'établissement scolaire, et potentiellement, à une échelle plus large au niveau régional voire national.
    Langage : HTML, CSS, Php, MySql

  • Test d'Admission (En cours de construction)


    Le Projet du Test d'Admission, était de proposé à l'établissement lors de mon stage de 2nd année, un Test permettant de tester le niveau des élèves. Les réponses sont récupérés automatiquement à partir d'une base de données.
    Langage : HTML, CSS, PHP, MySql

  • Portfolio


    Tout développeur doit avoir un Portfolio. Le Portfolio est un élément regroupant le parcours d'une personne ainsi que les projets qu'il a pu réaliser. Le Portfolio a été réalisé pour la présentation de celui-ci pour l'épreuve E4 du BTS SIO.
    Langage : HTML, CSS
